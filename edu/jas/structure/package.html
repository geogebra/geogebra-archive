Basic structural interfaces.

This package contains interfaces for the most general algebraic structures used in JAS, e.g. RingElem and RingFactory. Some classes implement e.g. quotient rings or residue class rings, based only on the interfaces in this package. The Product class implements a finite product of other ring elements.
