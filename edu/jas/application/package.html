Groebner base application package.

This package contains classes with applications of Groebner bases such as ideal intersections, ideal quotients or ideal dimension are implemented in Ideal and SolvableIdeal. The rational function class Quotient uses package edu.jas.ufd to compute quotients of polynomials reduced to lowest terms. Class Residue provides polynomials residues modulo an ideal defined in ResidueRing. Comprehensive Groebner bases for polynomial rings over parameter rings are implemented in ComprehensiveGroebnerBaseSeq.
