----

Nullsoft Scriptable Install System

Copyright (C) 1995-2009 Contributors

The file
NSIS_2.46_UltraModernUI_1.00_2009-12-25.exe contains NSIS 2.46 with
UltraModernUI installer for Windows. It can be downloaded from
http://sourceforge.net/projects/ultramodernui/files/UltraModernUI/UltraModernUI%201.00b2-dev/

and installing. This eases the installation under Linux/Wine.
